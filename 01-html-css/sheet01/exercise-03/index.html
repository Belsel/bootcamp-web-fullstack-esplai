Bienvenidos al curso programame

En el curso programame aprenderás a programar desde cero, adquiriendo habilidades en desarrollo web, bases de datos y mucho más. Nuestro objetivo es brindarte una formación completa para que puedas convertirte en un desarrollador profesional.

Preguntas frecuentes

Índice

  • Temario
  • Inscripción

Temario

  • ¿Qué temario se da en el curso?

    En el curso se da todo el stack tecnológico tanto de front end como de back end necesario para desplegar un entorno moderno.

  • ¿Qué requisitos son necesarios para inscribirse?

    Para inscribirse en el curso, se requiere tener conocimientos básicos de programación y un entorno de desarrollo configurado.

Inscripción

  • ¿Cómo puedo inscribirme en el curso?

    Para inscribirte en el curso, puedes visitar nuestra página web oficial y completar el formulario de inscripción con tus datos personales y de contacto.

  • ¿Cuándo comienza el curso?

    El curso comienza el próximo mes. Te recomendamos revisar nuestro calendario de eventos para obtener la fecha exacta.
