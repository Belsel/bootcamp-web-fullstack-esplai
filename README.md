# Web fullstack course

## Description

This course reinforces learning of web technologies for modern architectures and good practices around the board.

## Goals

- Gain fundamental concepts of web development.
- Work good practices for collaborative work environement.
- Improve job seeking methods and presentation.
- Achieve self growth.

## Use

This repository contains all exercices for every subject of the course.
The subjects are cathegorized and divided by content blocks.
Blocks contain singular exercices.
